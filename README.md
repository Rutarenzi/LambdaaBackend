## Lambdaa dashboard Challenge

This backend challenge from Lambdaa internship challenge
## Prerequisities

1.install `node>=20`
2.install `postgresql`

## Technology 

```
1.Typescript
2.express
3.SWAGGER
```

## How to install
```
git clone "repository url"
yarn install
yarn migrate
yarn run start

```
